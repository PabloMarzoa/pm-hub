# Installation

To use the library, you need to meet the following requirements:

-   **Angular** project (compatible with version 20.0.0+)
-   **Tailwindcss** (version 3.4.x+)

## Steps

<br />
### 1 路 Install Dependencies

Install the `@pmarzoa` libraries:

```bash
npm install pm-fuel
```

### 2 路 Configure Angular (Global Styles)

Add the global styles and fonts to your main SCSS file (e.g., `styles.scss`). This handles font imports and core styles automatically.

```scss
/* Fonts */
@use 'pm-fuel/src/assets/styles/fonts.scss';

/* Icons */
@use 'pm-fuel/src/assets/styles/icons.scss';

/* Tailwind directives (if not already present) */
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### 3 路 Configure Assets (Optional but Recommended)

If you need to serve assets (like images or specific font files) directly, copy them to your assets folder.

Add this to your `angular.json` under `architect > build > options > assets`:

```json
{
	"glob": "**/*",
	"input": "./node_modules/pm-fuel/src/assets",
	"output": "/assets"
}
```

### 4 路 Select Your Theme

Add the EMI theme styles to your global style file:

```scss
@use 'pm-fuel/src/assets/styles/emi-colors.scss';
@use 'pm-fuel/src/assets/styles/emi-shapes.scss';
```

## Import Components

Import the specific components or pipes you need in your standalone components:

```typescript
import { PmdsCdkButtonComponent } from 'pm-fuel';

@Component({
	selector: 'app-my-component',
	standalone: true,
	imports: [PmdsCdkButtonComponent],
	template: `<pmds-cdk-button label="Click me"></pmds-cdk-button>`,
})
export class MyComponent {}
```
